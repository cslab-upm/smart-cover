## Smart Cover

Este dispositivo es una tapa inteligente para un telescopio que abre y cierra según la hora del día. El diseño se puede parametrizar para que cualquiera pueda adaptarlo al tamaño de su telescopio.

### Diseño mecánico

Diseños y Planos necesarios para construir la tapa del telescopio. Dentro podrá encontrarse una excel para calcular los parámetros necesarios de fabricación en base al diámetro de la tapa.

### PCB

Se incluye el plano electrónico y el archivo para construir la PCB.

### Firmware

Firmware necesario para controlar la tapa del telescopio robótico. Desarrollado para microcontrolador ESP32.

Para más detalles sobre los pasos a seguir, ver: [Instructables](https://www.instructables.com/Smart-Cover-of-a-Night-time-Telescope-for-Day-time/).
